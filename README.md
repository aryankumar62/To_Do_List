# 📝 To-Do List App

A simple and interactive To-Do List web app built using HTML, CSS, and JavaScript.

## 🚀 Features
- Add tasks
- Delete tasks
- Mark tasks as completed
- Data saved using localStorage
- Theme switch (Blue, Black, White)
- Live date & time display

## 🛠 Tech Used
- HTML
- CSS
- JavaScript

## 🎯 Purpose
This project helped me understand DOM manipulation, event handling, and localStorage.
